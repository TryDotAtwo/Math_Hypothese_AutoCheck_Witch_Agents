## Уточнённая формулировка (по отчётам v2–v5)

В проекте используются две эквивалентные нормировки. Связь с OEIS A007726 даётся в двух видах:

- **Формулировка A:** $\mathcal{P}_L = \prod_{0<j<k<L}(4 - 2\cos\frac{j\pi}{L} - 2\cos\frac{k\pi}{L}) = a(L)$, где $a(n) = \mathrm{A007726}(n)$ (спектр $2H$ из [[C-001]]).
- **Формулировка B (промпт):** $T = \frac{2}{L(L-1)}\prod_n (L-1 - E_n)$ (без тривиального $E=L-1$) $= a(L-1)$, с тригонометрическим произведением по $1\le j<k\le L-2$ и косинусами с знаменателем $L-1$. Источник: `data/raw/Hipo_1_update.md`.

Единственный на сегодня **строгий** вывод обеих формул — через отождествление $2H = \mathcal{L}(G_L)$ (лапласиан графа 2-магнонных конфигураций), изоморфизм $G_L \cong \mathrm{QAD}_{L-1}$ и теорему Кирхгофа (отчёты v2, v4). Вывод «только из уравнений Бете» не завершён: инвариантность по $\Delta$ опровергнута (v2, v3).

## Формулировка

> Рассмотрим XXX спин-цепочку Гейзенберга ([[C-001]]) для спина $1/2$ длины $L$ с **открытыми граничными условиями**. Рассмотрим множество собственных значений гамильтониана для сектора с 2 перевёрнутыми спинами (магнонами) при фиксированной конечной длине $L$.
>
> **Утверждение 1.** Произведение всех собственных значений в этом секторе является **целым числом**.
>
> **Утверждение 2.** Это произведение как функция длины цепочки совпадает с последовательностью [OEIS A007726](https://oeis.org/A007726).
>
> **Утверждение 3.** Произведение допускает замкнутую формулу:
> $$\mathcal{P}_L = \prod_{0 < j < k < L} \left( 4 - 2\cos\frac{j\pi}{L} - 2\cos\frac{k\pi}{L} \right)$$
> (возможно, с заменой $L \to L+1$ или $L-1$).

## Мотивация

Гипотеза связывает спектр квантовой интегрируемой системы (XXX-цепочка Гейзенберга) с комбинаторной целочисленной последовательностью. Произведение тригонометрических выражений по парам индексов неожиданно даёт целые числа, что указывает на глубокую алгебраическую структуру.

Последовательность A007726 связана с числом доминошных покрытий решёток и количеством остовных деревьев определённых графов, что намекает на комбинаторную интерпретацию спектра спин-цепочки.

## Заметки

- Ключевое тождество: $4 - 2\cos\alpha - 2\cos\beta = 4(\sin^2(\alpha/2) + \sin^2(\beta/2))$
- Целочисленность следует из того, что произведение — симметрическая функция от корней циклотомических полиномов, а значит — рациональное алгебраическое целое = целое число
- Сдвиг индексов: для формулы A имеем $a(L) = \mathcal{P}_L$; для величины $T$ из промпта $a(L-1) = T$.
