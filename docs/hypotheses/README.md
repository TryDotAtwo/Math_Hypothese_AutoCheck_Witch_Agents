# Гипотезы

Здесь хранятся карточки гипотез — центральные объекты исследования.

## Жизненный цикл гипотезы

```
Наблюдение → Формулировка → Проверка → Доказательство / Опровержение
   (I-)         (H-)          (E-)         (P-)
```

## Как оформить гипотезу

1. Скопируй `_template.md` → `H-{номер}_{краткое_название}.md`
2. Сформулируй точно и однозначно (LaTeX)
3. Укажи мотивацию: почему это важно или интересно
4. Определи частные случаи для проверки
5. Свяжи с концепциями `[[C-XXX]]` и другими гипотезами `[[H-XXX]]`

## Критерии хорошей гипотезы

- **Фальсифицируемость** — можно проверить вычислительно
- **Точность** — однозначная математическая формулировка
- **Нетривиальность** — не следует напрямую из известных результатов
- **Связность** — связана с другими гипотезами или теоремами

## Нумерация

Номера присваиваются последовательно: H-001, H-002, ...
Номера никогда не переиспользуются, даже если гипотеза опровергнута.
