Consider the XXX spin chain Hamiltonian for spin chain with spin 1/2 at each site and length L, but with open boundary conditions (i.e. we drop the term P_{L,1}). Write the Bethe ansatz equations for it. Consider the set of this Hamiltonian's eigenvalues for 2 flipped spins at fixed finite length. Starting from the Bethe ansatz equations, show that the product of these eigenvalues is an integer, and that it matches (as a function of the spin chain length) the sequence https://oeis.org/A007726. Show that we have the following nice representation for this product: Product_{0<j<k<L} (4 - 2*cos(j*Pi/L) - 2*cos(k*Pi/L))   (possibly with L replaced by L+1 or L-1).
