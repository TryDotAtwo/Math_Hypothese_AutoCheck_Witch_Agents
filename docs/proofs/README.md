# Доказательства

Полные доказательства, попытки и наброски.

## Философия

Каждая попытка доказательства ценна, даже неудачная. Неудачные подходы показывают, где скрыта сложность задачи, и помогают найти правильный путь.

## Статусы

- **🟡 in-progress** — работа идёт
- **🔵 complete** — доказательство завершено и проверено
- **🔴 failed** — подход не сработал (описано почему)

## Как оформить

1. Скопируй `_template.md` → `P-{номер}_{краткое_название}.md`
2. Укажи связь с гипотезой `[[H-XXX]]`
3. Опиши стратегию высокоуровнево
4. Пиши доказательство, отмечая пробелы как **[???]**
5. Если подход не сработал — опиши почему и какие альтернативы есть
