## Стратегия

Показать, что определитель гамильтониана в 2-магнонном секторе не зависит от параметра анизотропии $\Delta$, и вычислить его явно в свободно-фермионном пределе $\Delta = 0$, где уравнения анзаца Бете расцепляются.

## Используемые результаты

- [[C-001]] — определение XXX-цепочки с открытыми границами
- [[C-002]] — уравнения анзаца Бете для $M=2$ и их $\Delta$-обобщение (XXZ)

## Доказательство

### Шаг 1. Алгебраическая формулировка BAE

В переменных $z_j = e^{ik_j}$ уравнения анзаца Бете для XXZ-цепочки с параметром $\Delta$ записываются:
$$z_1^{2L} = S_\Delta(z_1, z_2) \cdot S_\Delta(z_1, z_2^{-1})$$
$$z_2^{2L} = S_\Delta(z_2, z_1) \cdot S_\Delta(z_2, z_1^{-1})$$

где S-матрица:
$$S_\Delta(z, w) = -\frac{zw - 2\Delta z + 1}{zw - 2\Delta w + 1}$$

При $\Delta = 1$ (XXX) это исходная задача. При $\Delta = 0$ (XX) — свободные фермионы.

Расчистка знаменателей даёт два полинома $P_1(z_1, z_2) = 0$, $P_2(z_1, z_2) = 0$ с целыми коэффициентами (зависящими от $\Delta$ и $L$).

### Шаг 2. Характеристический полином через результант

Вводим полином энергии:
$$P_E(E, z_1, z_2) = E - 4 + (z_1 + z_1^{-1}) + (z_2 + z_2^{-1}) = 0$$

Характеристический полином 2-магнонного сектора получается элиминацией $z_1, z_2$:
$$Q(E) = \operatorname{Res}_{z_1, z_2}(P_1, P_2, P_E)$$

Произведение всех собственных значений = $(-1)^N Q(0) = \det(H_{\text{2-mag}})$.

### Шаг 3. Инвариантность определителя при деформации $\Delta$ — **ОПРОВЕРГНУТА**

**Исходное утверждение:** $\det(H_{\text{2-mag}})$ не зависит от $\Delta$ при непрерывной деформации $\Delta: 1 \to 0$.

**Статус:** В отчётах v2 и v3 проведена **численная проверка**: при изменении $\Delta$ число нулевых собственных значений (нулевых мод) меняется; произведение по ненулевым собственным значениям **не сохраняется**. Таким образом, шаг 3 неверен, и вывод тождества с A007726 **только** через BAE и $\Delta$-деформацию в текущем виде невозможен. См. отчёты [v2](../reports/H-001_opus4_6_v2.md), [v3](../reports/H-001_opus4_6_v3.md), [v4](../reports/H-001_opus4_6_v4.md) §4.2.

### Шаг 4. Вычисление в свободно-фермионном пределе ($\Delta = 0$)

При $\Delta = 0$: $S_0(z,w) = -1$, и BAE расцепляются:
$$z_1^{2L} = 1, \quad z_2^{2L} = 1$$

Физические решения (исключая $z = \pm 1$): $z_j = e^{i\pi j / L}$, $j = 1, \ldots, L-1$.

Принцип Паули для фермионов требует $z_1 \neq z_2$, что при упорядочении даёт:
$$0 < j < k < L$$

Подставляя в формулу энергии:
$$E_{j,k} = 4 - 2\cos\frac{j\pi}{L} - 2\cos\frac{k\pi}{L}$$

Произведение:
$$\prod_k E_k \Big|_{\Delta=0} = \prod_{0 < j < k < L}\left(4 - 2\cos\frac{j\pi}{L} - 2\cos\frac{k\pi}{L}\right)$$

### Шаг 5. Целочисленность

Используя $1 - \cos\theta = 2\sin^2(\theta/2)$:
$$\mathcal{P}_L = \prod_{0<j<k<L} 4\left(\sin^2\frac{j\pi}{2L} + \sin^2\frac{k\pi}{2L}\right)$$

Множители — значения симметрических функций от корней циклотомических полиномов. Произведение является алгебраическим целым (замкнутость $\overline{\mathbb{Z}}$ под умножением) и рациональным (Галуа-инвариантность). Рациональное алгебраическое целое — целое число, $\mathcal{P}_L \in \mathbb{Z}$.

## Альтернативный путь (доказан)

Равенство $\mathcal{P}_L = a(L)$ и $T = a(L-1)$ (OEIS A007726) **строго выводится** без $\Delta$-деформации:

1. **$2H_{\text{ours}} = \mathcal{L}(G_L)$** в 2-магнонном секторе — лапласиан графа $G_L$ (вершины — пары $(x_1,x_2)$, рёбра — сдвиг одного магнона). Доказано прямым вычислением (v2, v3).
2. **$G_L \cong \mathrm{QAD}_{L-1}$** — изоморфизм с четвертью ацтекского алмаза (v2, v3).
3. **Теорема Кирхгофа:** $\prod_{\lambda_i \neq 0} \lambda_i = |V|\,\tau(G)$.
4. **Формула Кнута:** $\tau(\mathrm{QAD}_{n}) = \mathrm{A007726}(n)$ с известным тригонометрическим произведением.

Численная проверка: скрипт `verify_prompt_T.py`. Подробно: отчёты [v2](../reports/H-001_opus4_6_v2.md), [v4](../reports/H-001_opus4_6_v4.md), [v5](../reports/H-001_opus4_6_v5.md).
