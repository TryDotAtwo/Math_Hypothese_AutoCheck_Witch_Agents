# Эксперименты

Журнал вычислительных экспериментов — проверка гипотез на реальных графах.

## Принципы

1. **Воспроизводимость** — каждый эксперимент содержит ссылку на код и параметры
2. **Документирование** — даже отрицательный результат полезен
3. **Связность** — каждый эксперимент привязан к гипотезе

## Как создать

1. Скопируй `_template.md` → `E-{номер}_{краткое_название}.md`
2. Укажи цель и связь с гипотезой
3. Задокументируй параметры и методологию
4. Запиши результаты и выводы
5. Если найден контрпример — опиши его максимально подробно
