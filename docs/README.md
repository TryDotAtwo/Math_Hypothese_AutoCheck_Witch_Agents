# Docs — Долгосрочная память проекта

Эта папка — центральное хранилище знаний проекта, построенное по принципам Zettelkasten.

## Принципы

1. **Атомарность** — одна заметка = одна идея, одно определение, одна гипотеза
2. **Связность** — каждая заметка ссылается на связанные через `[[ID]]`
3. **Постоянство** — заметки не удаляются; если идея устарела, она помечается статусом
4. **Эволюция** — заметки обновляются по мере развития понимания

## Структура

### Ядро знаний

| Папка | Назначение | Шаблон |
|-------|-----------|--------|
| `hypotheses/` | Гипотезы и конъектуры | `hypotheses/_template.md` |
| `concepts/` | Определения, теоремы, леммы | `concepts/_template.md` |
| `proofs/` | Доказательства и попытки | `proofs/_template.md` |
| `experiments/` | Вычислительные эксперименты | `experiments/_template.md` |
| `insights/` | Наблюдения и связи между идеями | `insights/_template.md` |
| `references/` | Источники: статьи, книги | `references/_template.md` |
| `journal/` | Ежедневный/еженедельный дневник | Свободная форма |

### Входная точка (inbox)

| Папка | Назначение |
|-------|-----------|
| `inbox/raw/` | **Сюда кидать всё сырое** — голые md-файлы, переписки с LLM, наброски. Агент сам оформит в карточки и перенесёт. |
| `inbox/processed/` | Обработанные файлы (после работы агента) |

### Навигация и организация

| Папка | Назначение |
|-------|-----------|
| `domains/` | Разделение по типам графов (деревья, планарные, раскраски...) |
| `dialogues/` | Архив диалогов с LLM (включая неудачные) |
| `models/` | Рабочие пространства AI-моделей + синтез результатов |

## Как создавать заметки

1. Скопируй `_template.md` из нужной папки
2. Назови файл по схеме: `{ID}_{краткое_название}.md` (например `H-001_chromatic_bound.md`)
3. Заполни все поля шаблона
4. Добавь ссылки на связанные заметки `[[C-003]]`, `[[H-012]]`
5. Обнови `index.md` в корне docs

## Статусы заметок

| Статус | Значение |
|--------|----------|
| `🟢 active` | Актуальная заметка |
| `🔵 proved` | Доказано |
| `🔴 disproved` | Опровергнуто (контрпример найден) |
| `🟡 in-progress` | В работе |
| `⚪ archived` | Устарело, но сохранено для истории |

## Поиск и навигация

- **`index.md`** — мастер-индекс всех заметок
- **`domains/`** — навигация по типам графов (деревья, планарные, ...)
- **ID-ссылки** `[[H-001]]` — для связи между заметками
- **Теги** `#tags` — для тематической группировки
- **`models/`** — анализы от разных AI-моделей по каждой гипотезе
