# [C-002] Анзац Бете для открытых цепочек

**Тип:** теорема / метод
**Область:** спектральная теория, математическая физика
**Дата создания:** 2026-03-04

## Определение

> Для XXX спин-$1/2$ цепочки длины $L$ с открытыми граничными условиями ([[C-001]]) собственные состояния в секторе $M$ магнонов параметризуются набором быстрот (rapidities) $\{\lambda_1, \ldots, \lambda_M\}$.
>
> Открытые границы действуют как отражающие стенки: магнон пробегает расстояние $2L$ (туда и обратно), а рассеяние учитывает «зеркальные» образы других магнонов.
>
> **Для $M=2$ магнонов** уравнения анзаца Бете (BAE) имеют вид:
> $$\left(\frac{\lambda_1 - i/2}{\lambda_1 + i/2}\right)^{2L} = \frac{\lambda_1 - \lambda_2 - i}{\lambda_1 - \lambda_2 + i} \cdot \frac{\lambda_1 + \lambda_2 - i}{\lambda_1 + \lambda_2 + i}$$
> $$\left(\frac{\lambda_2 - i/2}{\lambda_2 + i/2}\right)^{2L} = \frac{\lambda_2 - \lambda_1 - i}{\lambda_2 - \lambda_1 + i} \cdot \frac{\lambda_2 + \lambda_1 - i}{\lambda_2 + \lambda_1 + i}$$
>
> Энергия состояния:
> $$E = \frac{1}{\lambda_1^2 + 1/4} + \frac{1}{\lambda_2^2 + 1/4} = 4 - 2\cos k_1 - 2\cos k_2$$
> где $\lambda_j = \frac{1}{2}\cot(k_j/2)$.

## Интуиция

В замкнутой цепочке магнон «наматывает круг» длины $L$ и набирает фазы рассеяния от других магнонов. В открытой цепочке магнон пробегает длину $2L$ (туда-обратно), и рассеивается не только на реальных магнонах, но и на их зеркальных отражениях. Поэтому показатель степени $2L$, а правая часть содержит два множителя: от прямого рассеяния ($\lambda_1 - \lambda_2$) и от зеркального ($\lambda_1 + \lambda_2$).

## Примеры

1. **$M=1$, свободный магнон:** BAE вырождается в $z^{2L} = 1$, давая $k_j = \pi j / L$, $j = 1, \ldots, L-1$.
2. **$M=2$, XX-предел ($\Delta=0$):** S-матрица становится тривиальной ($S=-1$), уравнения расцепляются, квантовые числа квантуются независимо с $0 < j < k < L$.

## Ключевые свойства

- Экспонента $2L$ (а не $L$) — следствие отражения от открытых границ
- Зеркальные фазы ($\lambda_j + \lambda_k$) — уникальная черта открытых систем
- В переменных $z_j = e^{ik_j}$: S-матрица $S(z,w) = -\frac{zw - 2z + 1}{zw - 2w + 1}$
- XXZ-обобщение: параметр анизотропии $\Delta$ входит через $S_\Delta(z,w) = -\frac{zw - 2\Delta z + 1}{zw - 2\Delta w + 1}$
