# Концепции

Определения, теоремы, леммы и свойства — строительные блоки теории.

## Организация

Каждая заметка содержит ровно одну концепцию: одно определение, одну теорему или одну лемму. Это позволяет точно ссылаться из гипотез и доказательств.

## Типы заметок

- **Определение** — формальное определение объекта или свойства
- **Теорема** — доказанный результат (со ссылкой на доказательство)
- **Лемма** — вспомогательный результат
- **Свойство** — наблюдение или следствие

## Как создать

1. Скопируй `_template.md` → `C-{номер}_{краткое_название}.md`
2. Заполни формальное определение (LaTeX)
3. Добавь интуитивное объяснение
4. Приведи примеры и контрпримеры
5. Свяжи с другими концепциями и гипотезами
