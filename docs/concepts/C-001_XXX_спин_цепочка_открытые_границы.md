# [C-001] XXX спин-цепочка Гейзенберга с открытыми граничными условиями

**Тип:** определение
**Область:** спектральная теория, математическая физика
**Дата создания:** 2026-03-04

## Определение

> **XXX спин-$1/2$ цепочка Гейзенберга** длины $L$ с открытыми граничными условиями задаётся гамильтонианом:
> $$H = \sum_{j=1}^{L-1} \left( \frac{1}{4} - \vec{S}_j \cdot \vec{S}_{j+1} \right)$$
> где $\vec{S}_j = \frac{1}{2}\vec{\sigma}_j$ — операторы спина на $j$-м узле, $\vec{\sigma}_j$ — матрицы Паули.
>
> Эквивалентная запись через матрицы Паули:
> $$H = \frac{1}{2}\sum_{n=1}^{L-1} \left(1 - \vec{\sigma}_n \cdot \vec{\sigma}_{n+1}\right)$$
>
> «Открытые границы» означают, что в сумме **нет** члена $\vec{S}_L \cdot \vec{S}_1$ (в отличие от периодических граничных условий).

## Интуиция

Квантовая цепочка из $L$ узлов, на каждом из которых сидит спин $1/2$. Соседние спины взаимодействуют изотропно (одинаково по всем осям — поэтому «XXX»). Открытые границы означают, что цепочка не замкнута в кольцо.

Полностью поляризованное состояние (все спины вверх) $|\uparrow\uparrow\cdots\uparrow\rangle$ является собственным с нулевой энергией. «Перевёрнутые» спины — магноны — играют роль квазичастиц; их динамика описывается анзацем Бете ([[C-002]]).

## Примеры

1. **$L=2$:** $H = \frac{1}{4} - \vec{S}_1 \cdot \vec{S}_2$. Два состояния: синглет (энергия 1) и триплет (энергия 0).
2. **$L=3$, 1 магнон:** три состояния с энергиями $\epsilon_j = 2 - 2\cos(\pi j/3)$, $j=1,2$.

## Ключевые свойства

- Эквивалентная запись через операторы перестановки: $H_{\text{prompt}} = \sum_{k=1}^{L-1} P_{k,k+1}$ связана с $H$ выше равенством $H_{\text{prompt}} = (L-1) - 2H$ (подробно: отчёт [H-001_opus4_6_v4](../reports/H-001_opus4_6_v4.md), §2.1).
- Интегрируемая система — решается точно через анзац Бете
- $SU(2)$-симметрия: $[H, \vec{S}_{\text{tot}}] = 0$
- Спектр разбивается на секторы по числу магнонов $M = 0, 1, \ldots, L/2$
- Связь с дискретным лапласианом графа-пути $P_L$ через свободно-фермионный предел ($\Delta \to 0$)
