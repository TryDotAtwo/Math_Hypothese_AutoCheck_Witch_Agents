# Источники

Статьи, книги, препринты и другие материалы.

## Как оформить

1. Скопируй `_template.md` → `R-{номер}_{фамилия_год}.md`
2. Заполни библиографическую ссылку
3. Выпиши ключевые результаты и идеи
4. Свяжи с нашими гипотезами и концепциями
5. Добавь личные заметки
