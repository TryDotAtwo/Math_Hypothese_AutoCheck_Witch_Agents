# Инсайты

Наблюдения, интуиции и неожиданные связи — самое ценное в исследовании.

## Что записывать

- Паттерны, замеченные в экспериментах
- Неожиданные связи между концепциями
- Аналогии с другими областями математики
- Интуитивные догадки, даже если пока не можешь их формализовать
- «Странные» результаты, которые не укладываются в текущую картину

## Почему это важно

Инсайты — это семена будущих гипотез. Многие великие теоремы начинались с простого наблюдения. Записывай всё, даже если кажется тривиальным — связи могут проявиться позже.

## Как создать

1. Скопируй `_template.md` → `I-{номер}_{краткое_название}.md`
2. Опиши наблюдение максимально конкретно
3. Подумай, почему это важно
4. Свяжи с другими заметками
5. Когда инсайт созреет — оформи как гипотезу
