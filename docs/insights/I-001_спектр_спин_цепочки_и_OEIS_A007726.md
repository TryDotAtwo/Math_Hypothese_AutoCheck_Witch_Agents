# [I-001] Инсайт: Произведение собственных значений спин-цепочки = комбинаторная последовательность

**Дата:** 2026-03-04
**Источник:** наблюдение / формулировка задачи из inbox

## Наблюдение

> Произведение всех собственных значений в 2-магнонном секторе XXX-цепочки ([[C-001]]) при открытых граничных условиях оказывается **целым числом**, причём совпадающим с последовательностью [OEIS A007726](https://oeis.org/A007726).
>
> Последовательность A007726 связана с числом доминошных покрытий решёток и остовных деревьев. Это указывает на неожиданный мост между:
> - **квантовой механикой** (спектр интегрируемого гамильтониана)
> - **комбинаторикой** (целочисленные последовательности, тайлинги)
> - **спектральной теорией графов** (собственные значения дискретного лапласиана графа-пути)

## Почему это важно

- Целочисленность произведения неочевидна из формулы — тригонометрические выражения с иррациональными значениями «чудом» сокращаются
- Формула $\prod_{0<j<k<L}(4 - 2\cos\frac{j\pi}{L} - 2\cos\frac{k\pi}{L})$ содержит в себе $\binom{L-1}{2}$ множителей, каждый иррациональный — но произведение целое
- Связь между интегрируемыми спин-цепочками и комбинаторными объектами может открывать новые тождества для последовательностей OEIS
- Механизм целочисленности (алгебраические целые + симметрическая функция от корней циклотомических полиномов = рациональное число) может обобщаться на другие секторы ($M > 2$)
