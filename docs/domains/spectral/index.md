# Домен: Спектральная теория графов

**Описание:** Изучение графов через собственные значения их матриц (смежности, Лапласиана, нормализованного Лапласиана).
**Ключевые объекты:** спектр, спектральный радиус, алгебраическая связность (число Фидлера), энергия графа

## Основные определения

| ID | Концепция | Описание |
|----|-----------|----------|
| C-001 | XXX спин-цепочка Гейзенберга (открытые границы) | Квантовая спин-$1/2$ цепочка длины $L$ с гамильтонианом $H = \sum_{j=1}^{L-1}(\frac{1}{4} - \vec{S}_j \cdot \vec{S}_{j+1})$ |
| C-002 | Анзац Бете для открытых цепочек | Метод точного решения: BAE с показателем $2L$ и зеркальными фазами |

| Теорема | Формулировка | Ссылка |
