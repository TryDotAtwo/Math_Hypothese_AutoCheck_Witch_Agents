# Домены — Разделение по типам графов и областям

Гипотезы могут быть о совершенно разных объектах: деревьях, планарных графах, гиперграфах, раскрасках... Домены — это навигационный слой, который позволяет работать в рамках одной темы, не теряя связей с остальными.

## Как это работает

```
                    ┌──────────────────────┐
                    │  docs/hypotheses/    │  ← Все гипотезы живут здесь
                    │  H-001, H-002, ...   │     (единое пространство ID)
                    └──────────┬───────────┘
                               │
              ┌────────────────┼────────────────┐
              ▼                ▼                 ▼
     ┌──────────────┐ ┌──────────────┐ ┌──────────────┐
     │ domains/     │ │ domains/     │ │ domains/     │
     │ trees/       │ │ planar/      │ │ coloring/    │  ← Каждый домен
     │ index.md     │ │ index.md     │ │ index.md     │     собирает своё
     └──────────────┘ └──────────────┘ └──────────────┘
```

**Принцип:** гипотезы, концепции и доказательства хранятся централизованно (единая нумерация H-001, C-001...). Каждый домен — это **представление (view)**, которое собирает ссылки на всё, что относится к данному типу графов.

## Зачем

1. **Фокус** — работаешь над деревьями? Открой `domains/trees/index.md` и видишь всё релевантное
2. **Перекрёстные связи** — гипотеза может быть в нескольких доменах одновременно
3. **Обзор** — быстро понять, в какой области больше прогресса, а где пробелы

## Стартовые домены

| Домен | Папка | Описание |
|-------|-------|----------|
| Деревья и леса | `trees/` | Ациклические связные графы |
| Планарные графы | `planar/` | Графы, укладываемые на плоскости |
| Двудольные графы | `bipartite/` | Графы без нечётных циклов |
| Раскраски | `coloring/` | Хроматическое число, рёберные раскраски |
| Спектральная теория | `spectral/` | Собственные значения, спектральный радиус |
| Связность | `connectivity/` | Вершинная и рёберная связность, разрезы |
| Экстремальная ТГ | `extremal/` | Теорема Турана, задачи Рамсея |
| Случайные графы | `random/` | Модель Эрдёша-Реньи, пороговые функции |
| Ориентированные | `directed/` | Орграфы, турниры |

## Как добавить новый домен

1. Создай папку `domains/{name}/`
2. Скопируй `_domain_template/index.md` в новую папку
3. Заполни описание и начни добавлять ссылки
4. Добавь домен в таблицу в `taxonomy.md`

## Одна гипотеза — несколько доменов

Это нормально! Гипотеза о хроматическом числе планарных графов принадлежит и `coloring/`, и `planar/`. Укажи все домены в поле `Домены:` карточки гипотезы.
