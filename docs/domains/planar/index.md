## Ключевые теоремы (известные)

| Теорема | Формулировка | Ссылка |
|---------|-------------|--------|
| Формула Эйлера | $V - E + F = 2$ для связных планарных графов | |
| Теорема Куратовского | Граф планарен ⟺ не содержит подразбиения $K_5$ и $K_{3,3}$ | |
| Теорема о 4 красках | $\chi(G) \leq 4$ для любого планарного графа | |
| Следствие из Эйлера | $E \leq 3V - 6$ для планарных графов с $V \geq 3$ | |
