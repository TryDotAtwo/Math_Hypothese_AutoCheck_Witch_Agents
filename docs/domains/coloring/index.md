# Домен: Раскраски

**Описание:** Вершинные, рёберные и списочные раскраски графов. Хроматическое число и его оценки.

## Ключевые теоремы (известные)

| Теорема | Формулировка | Ссылка |
|---------|-------------|--------|
| Теорема Брукса | $\chi(G) \leq \Delta(G)$ для связных графов, не являющихся полными или нечётными циклами | |
| Теорема Визинга | $\Delta(G) \leq \chi'(G) \leq \Delta(G) + 1$ | |
| Жадная оценка | $\chi(G) \leq \Delta(G) + 1$ | |
