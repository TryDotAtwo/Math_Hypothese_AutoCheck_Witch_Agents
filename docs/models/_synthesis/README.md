# Синтез — Сравнение результатов моделей

Здесь хранятся сравнительные анализы: что разные модели сказали по одной и той же гипотезе, где они согласны, где расходятся, и что из этого полезно.

## Как оформить синтез

1. Скопируй `_template.md` → `H-XXX_synthesis.md`
2. Собери анализы всех моделей по данной гипотезе
3. Сравни подходы, найди консенсус и разногласия
4. Извлеки лучшие идеи в основную базу знаний
