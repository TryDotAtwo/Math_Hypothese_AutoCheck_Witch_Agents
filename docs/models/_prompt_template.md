# Шаблон промпта для анализа гипотезы

Используй этот шаблон (адаптируя под конкретную модель) для единообразного анализа.

---

## Базовый промпт

```
Ты — математик-исследователь в области теории графов.

Проанализируй следующую гипотезу:

**Гипотеза [H-XXX]:** {точная формулировка}

**Контекст:**
- Область: {домен(ы)}
- Мотивация: {откуда возникла}
- Уже проверено: {что известно}

**Задачи:**
1. Оцени правдоподобность гипотезы
2. Предложи подходы к доказательству (минимум 2 разных)
3. Проверь на стандартных классах: деревья, циклы, полные графы, двудольные, планарные
4. Предложи потенциальные контрпримеры или объясни, почему их быть не должно
5. Укажи связи с известными теоремами и открытыми проблемами

Будь строг и критичен. Если видишь ошибку в формулировке — укажи на неё.
```

## Расширенный промпт (для сложных гипотез)

```
... (базовый промпт) ...

Дополнительно:
6. Какие леммы потребуются для доказательства?
7. Есть ли обобщение этой гипотезы?
8. Есть ли усиление (более сильная версия)?
9. Как эта гипотеза связана с: {перечисли связанные гипотезы}
10. Оцени по шкале 1-10: насколько эта гипотеза нетривиальна?
```

## Промпт для поиска контрпримеров

```
Найди контрпример к следующей гипотезе (или докажи, что его нет):

**Гипотеза:** {формулировка}

Стратегия поиска:
1. Начни с малых графов (3-7 вершин)
2. Проверь экстремальные случаи (пустой граф, полный граф, звезда)
3. Попробуй «плохие» конструкции: графы Петерсена, Мычельского, Кнезера
4. Если контрпример не найден — объясни, почему его может не быть
```
