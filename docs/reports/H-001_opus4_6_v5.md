## 2. Канонические формулировки (после согласования)

### 2.1. Две связи с OEIS A007726

В проекте используются две эквивалентные нормировки гамильтониана:

- **$H_{\text{ours}}$** ([[C-001]]): $H = \sum_{j=1}^{L-1}\bigl(\frac{1}{4} - \vec{S}_j\cdot\vec{S}_{j+1}\bigr)$.
- **$H_{\text{prompt}}$** (из `data/raw/Hipo_1_update.md`): $H = \sum_{k=1}^{L-1} P_{k,k+1}$.  
  Связь: $H_{\text{prompt}} = (L-1) - 2 H_{\text{ours}}$ (см. отчёт v4, §2.1).

**Формулировка A (тригонометрическое произведение для спектра $2H_{\text{ours}}$):**

$$\mathcal{P}_L = \prod_{0<j<k<L}\left(4 - 2\cos\frac{j\pi}{L} - 2\cos\frac{k\pi}{L}\right) = a(L),$$

где $a(n) = \mathrm{A007726}(n)$. Подтверждено вычислительно для $L = 3, \ldots, 14$ (отчёты v2, v3; скрипты `investigate_h001_v2.py`, `investigate_advanced_v2.py`).

**Формулировка B (величина $T$ из промпта):**

$$T = \frac{2}{L(L-1)} \prod_{n:\ E_n \neq L-1} (L-1 - E_n) = \prod_{1\le j<k\le L-2}\left(4 - 2\cos\frac{j\pi}{L-1} - 2\cos\frac{k\pi}{L-1}\right) = a(L-1).$$

Здесь $E_n$ — собственные значения $H_{\text{prompt}}$ в 2-магнонном секторе; тривиальное $E = L-1$ исключено. Проверка: скрипт `verify_prompt_T.py`.

### 2.2. Строгий вывод (единственный на сегодня полный путь)

1. **$2 H_{\text{ours}} = \mathcal{L}(G_L)$** в 2-магнонном секторе — лапласиан графа $G_L$ (вершины — пары $(x_1,x_2)$, $1\le x_1<x_2\le L$; рёбра — сдвиг одного магнона). Доказано прямым вычислением матричных элементов (v2, v3).
2. **$G_L \cong \mathrm{QAD}_{L-1}$** — изоморфизм с четвертью ацтекского алмаза (явные координаты $(u,v) = (x_1-1, x_2-x_1-1)$) (v2, v3).
3. **Теорема Кирхгофа:** $\prod_{\lambda_i \neq 0} \lambda_i = |V|\,\tau(G)$.
4. **Формула Кнута:** $\tau(\mathrm{QAD}_{n}) = a(n) = \mathrm{A007726}(n)$ с тригонометрическим произведением по $0<j<k<n$.

Отсюда $T = a(L-1)$ и формула B; формула A получается тем же графом при корректной индексации (v4).

### 2.3. Вывод «только из уравнений Бете»

- **Пробел:** В [[P-001]] шаг 3 (инвариантность $\det(H)$ при деформации $\Delta$) **опровергнут численно** (v2, v3): при изменении $\Delta$ число нулевых мод меняется, произведение по ненулевым собственным значениям не сохраняется.
- **Итог:** Строгого вывода равенства $T = a(L-1)$ (или $\mathcal{P}_L = a(L)$) **исключительно из BAE** без графовой интерпретации на сегодня нет.

## 5. Итог

- **Доказано:** Равенства $\mathcal{P}_L = a(L)$ и $T = a(L-1)$ (OEIS A007726) выводятся строго через отождествление $2H_{\text{ours}}$ с лапласианом графа $G_L \cong \mathrm{QAD}_{L-1}$ и теорему Кирхгофа (и формулу Кнута).
- **Опровергнуто:** Утверждение о независимости произведения собственных значений 2-магнонного сектора от параметра $\Delta$ (шаг 3 P-001).
- **Открыто:** Строгий вывод «только из уравнений Бете» без графовой интерпретации; обобщение на $M>2$ (последовательности не отождествлены с OEIS).
