# Сводка: Δ-инвариантность определителя 2-магнонного сектора XXX-цепочки

**Модель:** XXX спин-цепочка Гейзенберга с открытыми границами (XXZ-деформация)
**Дата анализа:** 2026-03-04
**Гипотеза:** [[H-001]] — Произведение собственных значений 2-магнонного сектора = A007726
**Домен(ы):** `spectral`, `algebraic-geometry`

## 1. Формулировка задачи

Критический пробел в доказательстве [[P-001]]: требуется строгое обоснование **Δ-инвариантности** определителя гамильтониана в 2-магнонном секторе при непрерывной деформации параметра анизотропии от XXX-модели (Δ = 1) к XX-модели (Δ = 0).

**Конкретная математическая задача:**

Пусть H_2-mag(Δ) — гамильтониан в секторе с M = 2 магнонами для XXZ-цепочки с параметром анизотропии Δ ∈ [0, 1]. Доказать, что:

$$\det(H_{\text{2-mag}}(Δ)) = \det(H_{\text{2-mag}}(0)) = \det(H_{\text{2-mag}}(1))$$

При этом:
- При Δ = 0 (свободные фермионы): собственные значения известны явно: $E_{j,k} = 4 - 2\cos\frac{j\pi}{L} - 2\cos\frac{k\pi}{L}$
- При Δ = 1 (XXX-модель): собственные значения определяются нелинейными уравнениями анзаца Бете
- Произведение при Δ = 0 даёт последовательность A007726

Без доказательства Δ-инвариантности переход от XXX к XX остаётся эвристикой, а не строгим доказательством.

## 2. Что обнаружено

### 2.1 Структура задачи

Анализ показал, что задача сводится к изучению **результантной структуры** системы полиномиальных уравнений анзаца Бете (BAE).

**Ключевые объекты:**

1. **Уравнения BAE в переменных $z_j = e^{ik_j}$:**
   $$z_1^{2L} = S_Δ(z_1, z_2) \cdot S_Δ(z_1, z_2^{-1})$$
   $$z_2^{2L} = S_Δ(z_2, z_1) \cdot S_Δ(z_2, z_1^{-1})$$
   
   где S-матрица: $S_Δ(z, w) = -\frac{zw - 2Δz + 1}{zw - 2Δw + 1}$

2. **Полином энергии:**
   $$P_E(E, z_1, z_2) = E - 4 + (z_1 + z_1^{-1}) + (z_2 + z_2^{-1})$$

3. **Характеристический полином через результант:**
   $$Q_Δ(E) = \operatorname{Res}_{z_1, z_2}(P_1^Δ, P_2^Δ, P_E)$$

   где $P_1^Δ, P_2^Δ$ — полиномы BAE после расчистки знаменателей.

**Центральный вопрос:** Почему свободный член $Q_Δ(0) = (-1)^N \det(H_{\text{2-mag}}(Δ))$ не зависит от Δ?

### 2.2 Ключевые наблюдения

| # | Наблюдение | Статус | Ссылка |
|---|-----------|--------|--------|
| 1 | Определитель — это значение многомерного результанта трёх полиномов по переменным $z_1, z_2$ | подтверждено | [[P-001]], шаг 2 |
| 2 | Полиномы BAE $P_1^Δ, P_2^Δ$ аналитически зависят от Δ рационально (через линейное вхождение в числитель/знаменатель S-матрицы) | подтверждено | [[C-002]] |
| 3 | Результант как функция от коэффициентов полиномов — полиномиальная функция (универсальная формула) | подтверждено | Теория результантов |
| 4 | Δ-инвариантность НЕ очевидна из формул: результант зависит от Δ нетривиально | опровергнуто (ожидание инвариантности) | Анализ структуры |
| 5 | При деформации Δ собственные значения меняются непрерывно (аналитический возмущённый теорий), но могут пересекать ноль | предположительно | Общая теория возмущений |

### 2.3 Связи с известными результатами

**Алгебраическая геометрия спектральных кривых:**

Для интегрируемых систем ключевой объект — **спектральная кривая** (алгебраическая кривая, инвариантная под изоспектральными потоками). В контексте XXZ-цепочки:

- Спектральная кривая XXZ-модели изучена в работах по алгебраическому анзацу Бете (Nepomechie et al.)
- Существуют **Q-системы** для квантовых интегрируемых цепочек с компактными детерминантными выражениями для Q-функций
- Для A_m^{(1)} цепочек получены явные формулы для результантов

**Разделение переменных Склянина (Sklyanin's SoV):**

Метод E.K. Sklyanin (1988) для открытых цепочек предоставляет альтернативный подход к спектру:

- Конструирует разделённые переменные через функции A(u), B(u)
- Позволяет свести многомерную проблему к набору одномерных уравнений разделения
- Для XXX-цепочки даёт явные формулы для собственных значений через полюсы нормализованной функции Бейкера-Ахиезера

**Возможное применение:** метод SoV может дать более простую параметризацию собственных значений, где Δ-инвариантность станет очевидной.

## 4. Что НЕ удалось

**Не удалось найти:**

1. **Строгое доказательство отсутствия нулевых мод** при 0 ≤ Δ ≤ 1 для 2-магнонного сектора
   - Текущий аргумент о «границе вакуумного сектора» недостаточно строг
   - Требуется анализ спектрального зазора для всех Δ ∈ [0,1]

2. **Доказательство полиномиальности det(H) по Δ**
   - Результант как функция коэффициентов — полином, но зависимость коэффициентов BAE от Δ рациональна
   - Составная зависимость det(H)(Δ) требует дополнительного анализа

3. **Связь с работами Marinho/Nepomechie**
   - Найдены работы R. Nepomechie по алгебраическому анзацу Бете и Q-системам
   - Не найдены конкретные результаты о Δ-инвариантности определителей для открытых цепочек
   - Основной фокус Nepomechie — неоднородные TQ-уравнения, детерминантные формулы для корреляторов

4. **Прямое применение Sklyanin SoV**
   - Метод SoV даёт структуру спектра, но Δ-инвариантность произведения не очевидна
   - Требуется специальное исследование явных формул SoV для открытых XXX-цепочек

## 6. Математические инструменты для атаки на проблему

### 6.1 Результанты и теория исключения

**Многообразие решений BAE** как алгебраическое многообразие зависит от параметра Δ. Определитель — это значение результанта, который равен произведению значений полинома энергии в точках многообразия.

**Ключевая идея:** Исследовать, как меняется **кольцо координат** кольцевого многообразия BAE при деформации Δ.

### 6.2 Деформационная теория

**Подход:** Рассматривать семейство гамильтонианов H(Δ) как аналитическую деформацию. Если:
1. H(Δ) — аналитический оператор-значный функция по Δ
2. Спектр дискретен и непрерывен по Δ
3. Нет пересечений собственных значений с нулём при Δ ∈ [0,1]

то det(H)(Δ) — аналитическая функция без нулей, сохраняющая знак.

**Проблема:** Доказательство отсутствия пересечений с нулём (спектральный зазор).

### 6.3 Спектральные кривые и комплексная геометрия

Для XXZ-модели существует спектральная кривая, параметризующая решения BAE. При деформации Δ кривая меняется, но **число точек на кривой**, соответствующих физическим состояниям, может сохраняться.

**Гипотеза:** Произведение собственных значений связано с **дивизором** на спектральной кривой, инвариантным под деформацией.
