# H-001 Codex Report

## 1) Формулировка задачи

Проверяется гипотеза H-001 для open XXX spin-1/2 цепочки в секторе двух магнонов: произведение релевантных собственных значений совпадает с формулой

\[
\mathcal P_L=\prod_{0<j<k<L}\left(4-2\cos\frac{j\pi}{L}-2\cos\frac{k\pi}{L}\right),
\]

и связано с OEIS A007726 (остовные деревья quarter Aztec diamond).

## 2) Что обнаружено

- В текущем worktree отсутствуют карточки `docs/hypotheses/*`, `docs/concepts/*`, `docs/proofs/*`, `docs/insights/*` (есть только `.git`), поэтому контекст восстановлен из прошлых agent-транскриптов и внешних источников.
- Из проверенных транскриптов следует: главный пробел старых доказательств - недоказанный шаг про \(\Delta\)-инвариантность \(\det(H)\) в двухмагнонном секторе.
- По OEIS JSON (`A007726`) подтверждена точная формула:
  \[
  a(n)=\prod_{0<j<k<n}\left(4-2\cos\frac{j\pi}{n}-2\cos\frac{k\pi}{n}\right),
  \]
  и `OFFSET 1,3`.
- Следовательно для проектной записи \(\mathcal P_L\) корректное отображение индекса: \(n=L\) (без сдвига).

## 3) Что доказано / верифицировано сейчас

### 3.1 Надежно подтверждено по источникам

- A007726 действительно интерпретируется как число остовных деревьев quarter Aztec diamond.
- Формула произведения в A007726 совпадает по структуре с \(\mathcal P_L\), при \(n=L\).
- Целочисленность объясняется через теорему Кирхгофа (Matrix-Tree):
  \[
  \tau(G)=\frac{1}{|V|}\prod_{\lambda_i\neq 0}\lambda_i.
  \]
  Если последовательность считает остовные деревья, значение обязано быть целым.

## 4) Что НЕ удалось закрыть строго

- Полное строгое доказательство для всех \(L\) на уровне текущего запуска не получено.
- Шаг " \(\det(H_{2\text{-mag}})\) не зависит от \(\Delta\) " остается уязвимым без отдельного строгого обоснования (проблемы: возможные нулевые моды/резонансы, обратимость на пути деформации, корректность редукции сектора).
- Нужна аккуратная фиксация того, по какому именно множеству собственных значений берется произведение:
  - полный \(M=2\) сектор (возможен нулевой вклад и тривиальный нулевой продукт),
  - highest-weight часть,
  - или редуцированное регулярное подмножество, соответствующее формуле \(0<j<k<L\).

## 5) Строгая дорожная карта (следующий шаг к доказательству)

1. Зафиксировать нормировку open XXX гамильтониана и точный оператор в базисе \(|x,y\rangle\), \(1\le x<y\le L\).
2. Разделить подпространства (full \(M=2\), highest-weight, редуцированное регулярное) и явно выбрать одно целевое для H-001.
3. Построить coordinate Bethe ansatz для open-границ (framework Sklyanin), получить уравнения квантизации.
4. Доказать полноту решений в выбранном секторе (с учетом сингулярных/нулевых мод).
5. Выразить произведение энергий через симметрические полиномы/результант.
6. Свести к тригонометрическому произведению вида OEIS A007726.
7. Финально зафиксировать соответствие \(n=L\) и нормировку без неоднозначностей.
