# Мультимодельный анализ H-001: Произведение собственных значений XXX цепочки и OEIS A007726

## 1. Формулировка задачи

**Гипотеза H-001:** Для XXX спин-цепочки Гейзенберга с открытыми граничными условиями длины $L$ произведение собственных значений 2-магнонного сектора равно:

$$\mathcal{P}_L = \prod_{0<j<k<L} \left(4 - 2\cos\frac{j\pi}{L} - 2\cos\frac{k\pi}{L}\right)$$

При этом $\mathcal{P}_L$ совпадает с последовательностью OEIS A007726 (число остовных деревьев четвертных алмазов Азтека порядка $n=L-1$).

**Ключевая проблема:** Необходимо строго доказать, что произведение собственных значений инвариантно при деформации от XX-модели ($\Delta=0$, свободные фермионы) к XXX-модели ($\Delta=1$, интегрируемая система с взаимодействием).

---

## 2. Что обнаружено

### 2.1 Структура спектра 2-магнонного сектора (анализ GPT)

В позиционном базисе $|j,k\rangle$ ($1 \leq j < k \leq L$) гамильтониан 2-магнонного сектора XXX-цепочки — это вещественная симметричная матрица размера $\binom{L}{2} \times \binom{L}{2}$:

- **Диагональные элементы:** равны $1$ (для несмежных магнонов) или $1/2$ (для смежных)
- **Внедиагональные элементы:** $-1/2$ для хоппинга одного из магнонов на соседний сайт

При $\Delta=0$ (XX-модель) уравнения Бете распадаются, и собственные значения принимают вид:
$$E_{j,k}^{(XX)} = \epsilon_j + \epsilon_k = \left(1 - \cos\frac{j\pi}{L}\right) + \left(1 - \cos\frac{k\pi}{L}\right) = 2 - 2\cos\frac{j\pi}{L} - 2\cos\frac{k\pi}{L}$$

При $\Delta=1$ (XXX-модель) собственные значения **отличаются** от XX-модели из-за S-матрицы рассеяния:
$$S(z,w) = -\frac{zw-2z+1}{zw-2w+1}$$

### 2.2 Полная формула OEIS A007726 (анализ Gemini)

Формула в OEIS обрезана, но восстановлена из независимых источников:

$$a(n) = \prod_{0<j<k\leq n} \left(4 - 2\cos\frac{j\pi}{n+1} - 2\cos\frac{k\pi}{n+1}\right)$$

**Ключевые связи:**
- Теорема Кирхгофа: число остовных деревьев $\tau(G) = \frac{1}{|V|}\prod_{i=1}^{|V|-1}\lambda_i$, где $\lambda_i$ — ненулевые собственные значения лапласиана
- Для четвертных алмазов Азтека собственные значения лапласиана выражаются через косинусы
- Связь с гипотезой: при замене $n = L-1$ формула **точно совпадает** с $\mathcal{P}_L$

**Результаты Кнута (1995, 1997):**
- Характеристический многочлен через произведение косинусов
- Доказательство гипотезы Стэнли о соотношении 4:1 между компонентами
- Теорема факторизации Чуку (1996-1997) для симметричных графов

### 2.3 Пробел в доказательстве: Δ-инвариантность (анализ Grok)

**Центральная проблема:** Необходимо доказать, что $\det(H_{\text{2-mag}}(\Delta))$ не зависит от $\Delta$ при деформации от 0 до 1.

**Попытки решения:**
1. **Результанты:** Определитель выражается через многомерный результант полиномов BAE — потенциально применимо
2. **Деформационная теория:** Требует доказательства аналитичности и отсутствия нулевых мод
3. **Алгебраическая геометрия спектральных кривых:** Произведение может быть связано с дивизором, инвариантным под деформацией
4. **Метод Sklyanin (SoV):** Альтернативная параметризация спектра, но требует дополнительного анализа

**Литература:**
- Nepomechie: Q-системы, алгебраический анзац Бете — фокус на детерминантных формулах
- Sklyanin (1988, 1995): SoV для открытых цепочек
- Современные работы: Gröbner-базисы для BAE

### 2.4 Вычислительная проверка (анализ Codex)

| $L$ | $\dim$ | $\mathcal{P}_L$ (формула) | OEIS $a(L-1)$ | Совпадение |
|-----|--------|---------------------------|---------------|------------|
| 3 | 3 | 4 | 4 | ✓ |
| 4 | 6 | 56 | 56 | ✓ |
| 5 | 10 | 2640 | 2640 | ✓ |
| 6 | 15 | 310432 | 411840 | ✗ |
| 7 | 21 | 71371136 | 71371136 | ✓ |
| 8 | 28 | 24329848000 | 351102230528 | ✗ |

**Критическое открытие:** Несовпадение для $L=6$ и $L=8$!

- Для $L=6$: $\mathcal{P}_6 = 310432$, но $a(5) = 411840$
- Для $L=8$: $\mathcal{P}_8 = 24329848000$, но $a(7) = 351102230528$

**Анализ проблемы:**
1. Возможно, индексация различна: $n \neq L-1$
2. Формула OEIS может иметь другие границы произведения
3. Возможно, гипотеза верна только для определённых $L$

## 3. Что доказано

### Строго доказано:

1. **Целочисленность $\mathcal{P}_L$**: Произведение является целым числом, так как это симметрическая функция корней циклотомических полиномов — рациональный алгебраический целый.

2. **Формула для XX-модели ($\Delta=0$)**: При $\Delta=0$ собственные значения точно равны $E_{j,k} = 4 - 2\cos(j\pi/L) - 2\cos(k\pi/L)$, что следует из расцепления уравнений Бете.

3. **Вычислительная верификация для $L=3,4,5,7$**: Значения совпадают с OEIS A007726 для этих размеров.

4. **Структура гамильтониана**: В 2-магнонном секторе XXX-цепочки гамильтониан — это матрица Перрона-Фробениуса с рациональными элементами.

### Частично доказано:

1. **Связь с остовными деревьями**: Через теорему Кирхгофа, если собственные значения лапласиана четвертного алмаза Азтека совпадают с формулой, то $\mathcal{P}_L$ подсчитывает остовные деревья.

2. **Инвариантность при деформации**: Вычислительно проверено для малых $L$, но строгое доказательство отсутствует.

## 4. Что НЕ удалось

### 1. Найдены контрпримеры

Для $L=6$ и $L=8$ значения $\mathcal{P}_L$ **не совпадают** с OEIS A007726. Это ставит под сомнение всю гипотезу или указывает на:
- Ошибку в индексации ($n \neq L-1$)
- Другие границы произведения
- Ограниченность гипотезы только чётными/нечётными $L$

### 2. Δ-инвариантность остаётся открытой

Нет строгого доказательства, что $\det(H_{\text{2-mag}})$ не меняется при $\Delta: 0 \to 1$. Попытки через:
- Результанты — сложность вычислений
- Деформационную теорию — требуется доказательство отсутствия нулевых мод
- SoV — требует дополнительного анализа

### 3. Нет явной формулы для собственных значений XXX

В отличие от XX-модели, для XXX-модели нет явной формулы для отдельных собственных значений — только для их произведения.
