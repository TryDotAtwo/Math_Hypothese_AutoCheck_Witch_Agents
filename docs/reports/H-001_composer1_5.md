# Сводка: [H-001] Произведение собственных значений 2-магнонного сектора XXX-цепочки = A007726

**Модель:** Composer (Cursor)

**Гипотеза:** [[H-001]]
**Домен(ы):** `spectral`

## 1. Формулировка задачи

Рассмотрим XXX спин-$1/2$ цепочку Гейзенберга длины $L$ с открытыми граничными условиями ([[C-001]]). В 2-магнонном секторе (секторе с двумя перевёрнутыми спинами) имеется $\binom{L-1}{2}$ собственных значений гамильтониана.

Гипотеза утверждает три взаимосвязанных факта:

1. **Целочисленность произведения**: Произведение всех собственных значений в секторе является целым числом
2. **Связь с OEIS A007726**: Это произведение совпадает с последовательностью [OEIS A007726](https://oeis.org/A007726) (число остовных деревьев quarter Aztec diamonds)
3. **Явная формула**: Произведение даётся выражением $\mathcal{P}_L = \prod_{0 < j < k < L} \left( 4 - 2\cos\frac{j\pi}{L} - 2\cos\frac{k\pi}{L} \right)$

Последовательность A007726 связана с комбинаторными объектами: доминошными покрытиями прямоугольных решёток и остовными деревьями определённых графов.

## 2. Что обнаружено

### 2.1 Структура задачи

Гипотеза представляет собой нетривиальный мост между тремя областями математики:

- **Квантовая интегрируемость**: XXX-цепочка решается точно через анзац Бете с модификациями для открытых границ ([[C-002]])
- **Спектральная теория**: собственные значения выражаются через корни циклотомических полиномов
- **Комбинаторика**: неожиданная целочисленность указывает на скрытую дискретную структуру

Ключевой механизм — непрерывная деформация параметра анизотропии $\Delta$ от XXX-случая ($\Delta = 1$) к свободно-фермионному пределу ($\Delta = 0$), где уравнения анзаца Бете расцепляются.

### 2.3 Связи с известными результатами

- **Анзац Бете для открытых цепочек**: Уравнения модифицированы с экспонентой $2L$ и зеркальными фазами ([[C-002]])
- **Свободно-фермионный предел**: При $\Delta \to 0$ задача сводится к независимым фермионам с квантовыми числами $j,k = 1,\ldots,L-1$
- **Циклотомические полиномы**: Корни $\cos(j\pi/L)$ порождают симметрические функции, объясняющие целочисленность
- **OEIS A007726**: Комбинаторная последовательность, связанная с тайлингами Aztec diamonds

## 3. Что доказано

### 3.1 Полностью доказанные утверждения

> **Целочисленность произведения через алгебраическую структуру**: Произведение $\mathcal{P}_L$ является рациональным алгебраическим целым числом. Выражение $4 - 2\cos\alpha - 2\cos\beta = 4(\sin^2(\alpha/2) + \sin^2(\beta/2))$ показывает, что каждый множитель — симметрическая функция от корней циклотомических полиномов. Замкнутость кольца алгебраических целых и Галуа-инвариантность гарантируют рациональность.

Метод: Анализ алгебраической структуры выражения. Ссылка: [[P-001]], раздел о целочисленности.

> **Формула собственных значений в свободно-фермионном пределе**: При $\Delta = 0$ собственные значения имеют вид $E_{j,k} = 4 - 2\cos(j\pi/L) - 2\cos(k\pi/L)$ для $0 < j < k < L$.

Метод: Расцепление уравнений анзаца Бете в XX-пределе. Ссылка: [[P-001]], шаг 4.

## 4. Что НЕ удалось

- **Строгое доказательство Δ-инвариантности**: Центральный пробел в доказательстве [[P-001]]. Аргумент о непрерывной деформации неполон — требуется доказательство отсутствия нулевых собственных значений при промежуточных $\Delta$ или полиномиальности определителя по $\Delta$.
- **Комбинаторная интерпретация**: Несмотря на численное совпадение с A007726, отсутствует прямая биекция между собственными значениями и комбинаторными объектами (доминошные покрытия, остовные деревья).
- **Обобщение на M>2 магнонов**: Не сформулирована аналогичная гипотеза для секторов с большим числом магнонов; неизвестны соответствующие OEIS-последовательности.
- **Эффективное вычисление для больших L**: Отсутствуют расчёты для L≥6 из-за сложности решения полных уравнений анзаца Бете.
