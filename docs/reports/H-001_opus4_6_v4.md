## 1. Постановка задачи (из промпта)

Рассматривается **XXX спин-цепочка** со спином $1/2$ на каждом узле, длина $L$, **открытые граничные условия**, с гамильтонианом

$$H = \sum_{k=1}^{L-1} P_{k,k+1},$$

где $P_{k,k+1}$ — оператор перестановки спинов на соседних узлах $k$ и $k+1$.

- Рассматривается сектор с **двумя перевёрнутыми спинами** (2 магнона). Собственные значения ограничения $H$ на этот сектор обозначаются $E_n$, $n = 1, 2, \ldots, L(L-1)/2$.
- Вводится величина  
  $$T = \frac{2}{L(L-1)} \prod_n (L-1 - E_n),$$  
  где произведение берётся по **всем** этим собственным значениям, **кроме тривиального**, равного $L-1$.
- **Требуется:** объяснить, как получить из уравнений Бете (Бете-анзац) **строгий** вывод равенства  
  $$T = \prod_{1\le j<k\le L-2} \left(4 - 2\cos\frac{j\pi}{L-1} - 2\cos\frac{k\pi}{L-1}\right),$$  
  так что $T$ как функция $L$ совпадает с последовательностью [OEIS A007726](https://oeis.org/A007726). Использовать известные результаты со ссылками; явно указать места, где вывод не удаётся довести до конца.

Ниже даётся эквивалентная формулировка в наших обозначениях, связь с [[C-001]], [[C-002]], [[P-001]], затем вывод и граница его применимости.

## 2. Эквивалентность гамильтонианов и обозначений

### 2.1. Связь $H_{\text{prompt}} = \sum P_{k,k+1}$ с гамильтонианом из [[C-001]]

В [[C-001]] используется гамильтониан  
$$H_{\text{ours}} = \sum_{j=1}^{L-1} \left(\frac{1}{4} - \vec{S}_j\cdot\vec{S}_{j+1}\right).$$

Для спина $1/2$ на паре узлов $(k,k+1)$: $P_{k,k+1} = \frac{1}{2}(1 + \vec{\sigma}_k\cdot\vec{\sigma}_{k+1})$ и $\vec{S}_j = \frac{1}{2}\vec{\sigma}_j$, откуда $\vec{S}_k\cdot\vec{S}_{k+1} = \frac{1}{4}\vec{\sigma}_k\cdot\vec{\sigma}_{k+1} = \frac{1}{2}(P_{k,k+1} - \frac{1}{2})$. Поэтому  
$$\frac{1}{4} - \vec{S}_k\cdot\vec{S}_{k+1} = \frac{1}{2}(1 - P_{k,k+1}),$$  
и  
$$H_{\text{ours}} = \frac{1}{2}\sum_{k=1}^{L-1}(1 - P_{k,k+1}) = \frac{L-1}{2} - \frac{1}{2} H_{\text{prompt}}.$$

Следовательно,  
$$H_{\text{prompt}} = (L-1) - 2 H_{\text{ours}}.$$

В 2-магнонном секторе собственные значения связаны так: если $\mu$ — собственное значение $H_{\text{ours}}$, то $E = (L-1) - 2\mu$ — собственное значение $H_{\text{prompt}}$. В частности, тривиальное значение $E = L-1$ соответствует $\mu = 0$ (ферромагнитный вакуум в этом секторе).

### 2.2. Выражение $T$ через спектр $2H_{\text{ours}}$

Пусть $\lambda_1,\ldots,\lambda_N$ — все собственные значения ограничения $2H_{\text{ours}}$ на 2-магнонный сектор ($N = L(L-1)/2$). Один из них равен $0$, остальные положительны. Имеем $E_n = (L-1) - \lambda_n$, значит  
$$L-1 - E_n = \lambda_n.$$  
Исключая тривиальное $E = L-1$ (т.е. $\lambda = 0$), получаем  
$$\prod_{n:\ E_n \neq L-1} (L-1 - E_n) = \prod_{\lambda_n \neq 0} \lambda_n.$$

Поэтому  
$$T = \frac{2}{L(L-1)} \prod_{\lambda_n \neq 0} \lambda_n.$$

Это выражение мы используем для строгого вывода и численной проверки.

## 3. Уравнения Бете для открытой XXX-цепочки

Ниже приводятся уравнения Бете в форме, используемой в проекте ([[C-002]], [[P-001]]) и в стандартной литературе.

### 3.1. Открытая цепочка и анзац Бете

Для цепочки с открытыми границами собственные состояния в секторе с $M$ магнонами параметризуются наборами быстрот (rapidities) $\lambda_1,\ldots,\lambda_M$. Границы действуют как отражения, эффективная длина для фазы — $2L$ (туда и обратно). Ссылки: Sklyanin, «Boundary conditions for integrable quantum systems», J. Phys. A **21** (1988); Gaudin, «La fonction d’onde de Bethe» (1983).

### 3.2. Случай $M=2$

В переменных $z_j = e^{ik_j}$, $\lambda_j = \frac{1}{2}\cot(k_j/2)$, уравнения Бете для открытой XXX-цепочки имеют вид ([[C-002]]):

$$z_1^{2L} = S(z_1,z_2)\,S(z_1,z_2^{-1}), \qquad z_2^{2L} = S(z_2,z_1)\,S(z_2,z_1^{-1}),$$

где S-матрица  
$$S(z,w) = -\frac{zw - 2z + 1}{zw - 2w + 1}.$$

Энергия (в нормировке $H_{\text{ours}}$):  
$$E = \frac{1}{\lambda_1^2+1/4} + \frac{1}{\lambda_2^2+1/4} = 4 - 2\cos k_1 - 2\cos k_2.$$

После расчистки знаменателей получаются полиномы $P_1(z_1,z_2)$, $P_2(z_1,z_2)$. Характеристический полином по энергии получается элиминацией $z_1,z_2$ с полиномом энергии $P_E(E,z_1,z_2) = E - 4 + (z_1+z_1^{-1}) + (z_2+z_2^{-1})$ ([[P-001]], шаги 1–2).

## 4. Строгий вывод равенства $T = \prod_{1\le j<k\le L-2}(\ldots)$

### 4.1. Что удаётся вывести без пробелов

**Факт 1 (спектр как лапласиан графа).** В 2-магнонном секторе выполняется равенство операторов (доказано прямым вычислением матричных элементов в отчётах v2/v3):

$$2 H_{\text{ours}} = \mathcal{L}(G_L),$$

где $\mathcal{L}(G_L)$ — лапласиан графа $G_L$: вершины — пары $(x_1,x_2)$, $1\le x_1<x_2\le L$; рёбра — пары конфигураций, отличающиеся сдвигом одного магнона на соседний узел. Граф $G_L$ изоморфен четверти ацтекского алмаза $\mathrm{QAD}_{L-1}$ (явный изоморфизм в координатах $(u,v) = (x_1-1, x_2-x_1-1)$).

**Факт 2 (теорема Кирхгофа).** Для графа с $|V|$ вершинами число остовных деревьев  
$$\tau(G) = \frac{1}{|V|}\prod_{\lambda_i \neq 0} \lambda_i,$$  
где произведение берётся по ненулевым собственным значениям лапласиана. Ссылка: классическая матрично-дерево теорема (Kirchhoff).

**Факт 3 (Knuth, OEIS A007726).** Для четверти ацтекского алмаза порядка $n$ число остовных деревьев равно $a(n) = \mathrm{A007726}(n)$, причём  
$$a(n) = \prod_{0<j<k<n} \left(4 - 2\cos\frac{j\pi}{n} - 2\cos\frac{k\pi}{n}\right).$$  
Ссылки: Knuth, «Aztec Diamonds, Checkerboard Graphs, and Spanning Trees», J. Algebraic Combin. **6** (1997) 253–257; OEIS A007726.

У нас $|V(G_L)| = \binom{L}{2} = L(L-1)/2$. По Фактам 1–2:

$$\prod_{\lambda_n \neq 0} \lambda_n = |V(G_L)|\,\tau(G_L) = \frac{L(L-1)}{2}\,\tau(G_L).$$

По изоморфизму $G_L \cong \mathrm{QAD}_{L-1}$ и Факту 3: $\tau(G_L) = a(L-1)$. Подставляя в определение $T$:

$$T = \frac{2}{L(L-1)} \cdot \frac{L(L-1)}{2}\,a(L-1) = a(L-1).$$

С заменой индекса $n = L-1$ в формуле Кнута $a(n) = \prod_{0<j<k<n}(\ldots)$ получаем область $1\le j<k\le n-1 = L-2$ и знаменатель в косинусах $n = L-1$, т.е.:

$$T = \prod_{1\le j<k\le L-2} \left(4 - 2\cos\frac{j\pi}{L-1} - 2\cos\frac{k\pi}{L-1}\right).$$

Это и есть требуемое равенство. Вывод **строгий**, опирается на: (1) тождество $2H_{\text{ours}} = \mathcal{L}(G_L)$, (2) теорему Кирхгофа, (3) изоморфизм $G_L \cong \mathrm{QAD}_{L-1}$, (4) результат Кнута для A007726.

### 4.2. Вывод «напрямую из уравнений Бете»: что есть и чего не хватает

- **Из BAE при $\Delta=0$ (свободные фермионы):** при $S(z,w)=-1$ уравнения расцепляются, $z_j^{2L}=1$, откуда $k_j = \pi j/L$, $j=1,\ldots,L-1$, и при антисимметризации по двум частицам получается в точности формула с косинусами и произведением по $0<j<k<L$ ([[P-001]], шаг 4). Это даёт формулу для **другой** нормировки/индексации (наш $\mathcal{P}_L = a(L)$), а не для $T = a(L-1)$ в формулировке промпта.
- **Переход от $\Delta=1$ (XXX) к $\Delta=0$ (XX):** в [[P-001]] предполагалось, что определитель (произведение собственных значений) не зависит от $\Delta$. В отчётах v2/v3 это **опровергнуто** численно: при изменении $\Delta$ появляются/исчезают нулевые моды, произведение по ненулевым собственным значениям не сохраняется. Поэтому **полный вывод только из BAE** (без привлечения графа $G_L$ и Кирхгофа) в настоящее время **не восстановлен**: шаг «инвариантность по $\Delta$» не доказан и противоречит численным данным.

Итого: равенство $T = \prod_{1\le j<k\le L-2}(\ldots)$ строго выводится через спектр $2H_{\text{ours}}$, граф $G_L$ и A007726; вывод «исключительно из BAE» без графовой интерпретации остаётся открытым из-за отсутствия доказательства $\Delta$-инвариантности.

## 5. Численная проверка (скрипт `verify_prompt_T.py`)

Добавлен скрипт, который:

1. Строит $H_{\text{ours}}$ в 2-магнонном секторе (то же, что в `investigate_h001_v2.py`).
2. Вычисляет собственные значения $2H_{\text{ours}}$, берёт произведение по ненулевым $\lambda_n$.
3. Вычисляет $T = \frac{2}{L(L-1)} \prod_{\lambda_n\neq 0} \lambda_n$.
4. Вычисляет тригонометрическое произведение по формуле промпта (индексы $1\le j<k\le L-2$, косинусы с $\pi/(L-1)$).
5. Сравнивает с A007726(L-1).

Запуск: в корне проекта выполнить `python verify_prompt_T.py`. Ожидаемый результат: для всех проверенных $L$ значения $T$, тригонометрическое произведение и A007726(L-1) совпадают.
