# Рабочий процесс исследования

Пошаговое руководство по работе с гипотезами.

## Цикл исследования

### Фаза 1: Наблюдение

1. Работай с графами — генерируй, считай инварианты, визуализируй
2. Замечай паттерны: «кажется, что для всех деревьев выполняется X»
3. Записывай наблюдения в `docs/insights/`
4. Даже нечёткие интуиции стоит записать — они могут созреть позже

### Фаза 2: Формулировка гипотезы

1. Переведи наблюдение в точную математическую формулировку
2. Создай карточку в `docs/hypotheses/`
3. Определи область применимости: для каких графов?
4. Проверь тривиальные случаи вручную
5. Свяжи с известными результатами через `docs/concepts/`

### Фаза 3: Вычислительная проверка

1. Напиши предикат в Python:
   ```python
   def my_hypothesis(G, inv):
       return inv['alpha'] + inv['chi'] >= inv['n']
   ```
2. Запусти `test_hypothesis()` на стандартных семействах
3. Увеличивай размер графов: 5, 10, 20, 50, 100 вершин
4. Пробуй случайные графы с разными параметрами
5. Документируй в `docs/experiments/`

### Фаза 4a: Контрпример найден

1. Запиши контрпример подробно
2. Проанализируй: что особенного в этом графе?
3. Можно ли уточнить гипотезу, чтобы исключить контрпример?
4. Обнови статус гипотезы: 🔴 disproved или уточни формулировку
5. Часто контрпример — источник нового инсайта

### Фаза 4b: Гипотеза подтверждается

1. Попробуй найти доказательство
2. Начни с частных случаев: деревья, двудольные, планарные
3. Определи, какие свойства графа критичны
4. Документируй попытки в `docs/proofs/`
5. Неудачные попытки тоже ценны — они показывают, где сложность

### Фаза 5: Рефлексия

1. Запиши в `docs/journal/` что узнал
2. Обнови `docs/index.md`
3. Проверь связи: как новый результат влияет на другие гипотезы?
4. Подумай: какие новые вопросы возникли?

## Советы

- **Начинай с малого**: проверяй на графах с 3-7 вершинами
- **Доверяй, но проверяй**: даже если гипотеза прошла 10000 тестов, это не доказательство
- **Ищи структуру**: если гипотеза верна, попробуй понять *почему*
- **Связывай идеи**: самые интересные результаты на стыке разных тем
- **Записывай всё**: через месяц ты не вспомнишь детали без заметок
