# Исследовательский дневник

Свободная форма записей о ходе работы.

## Формат

Файлы именуются по дате: `YYYY-MM-DD.md`

Каждая запись может содержать:

- Над чем работал сегодня
- Какие идеи пришли в голову
- Что прочитал
- С какими трудностями столкнулся
- Планы на следующий сеанс работы

## Зачем вести дневник

1. **Контекст** — через месяц ты не вспомнишь, почему выбрал именно этот подход
2. **Паттерны** — помогает заметить повторяющиеся трудности
3. **Мотивация** — видеть прогресс за длительный период
4. **Рефлексия** — записывая мысли, ты их структурируешь
